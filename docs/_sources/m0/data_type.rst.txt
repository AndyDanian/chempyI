Data Types
----------

There are two data types in Python, structural and non--structural datas.



Non--Structural Data
^^^^^^^^^^^^^^^^^^^^
The non--structural data are variables which can save only one element, among this data type is:

* Integers
* Floats
* Booleans

.. code-block:: python
   :caption: Python's non--structural data.

   hello_world = 1
   PI = 3.14
   red_car = True


Structural Data
^^^^^^^^^^^^^^^
The structural data are variables which can save more one element, among this data type is:

* Chains of strings
* Lists
* Tuples
* Dictionaries
* Set

.. code-block:: python
   :caption: Python's structural data.

   # Chain example
   monday = 'Monday'
   # List example
   hello_world = list('Hello', 'World')
   hello_world = ['Hello', 'World']
   # Tuple example
   values = tuple(3.14, 300000)
   values = (3.14, 300000)
   values = 3.14, 300000
   # Dictionary example
   color_value = dict('color': 'red', 'value': 1000)
   color_value = {'color': 'red', 'value': 1000}
   # Set example
   colors = set('red', 'blue', 'yellow') 
   colors = {'red', 'blue', 'yellow'}
